### 使用 vscode 操作 git
    1. 创建 **本地仓库** 和 **远程仓库**
       * 创建远程库
       * 本地仓库要有.gitignore文件(没有就自己创建)，被忽略的文件
       * git init 初始化本地仓库
       * git add .xx 添加文件
       注意：/node_modules 代表根目录下的，你要放入仓库的文件
             node_modules 不加xxx则代表根目录及其子目录下的所有文件
    2. 将本地仓库代码**推送**到远程
    3. 如果**本地有变化**，推送到远程
    4. **远程有变化**，拉取到本地
    5. 新员工克隆远程仓库到本地
